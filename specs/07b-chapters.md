# SPEC 07b — Phase B: 訓練章コンテンツ（02 FLIGHT / 03 WEAPONS / 04 SURVIVAL / 05 COMM）

状態: **Draft** ／ 親: [SPEC-07](07-scroll-guide.md) ／ 前提: 07a Verified ／ 外部素材: 不要

## 1. 目的

遊び方説明の中核4章を、スクロール純関数の3D展開＋行単位のテキスト同期で実装する。
本SPECには**掲載コピー全文**と**t割当表**を含む（実装時の文言ブレを防ぐ）。

## 2. 要件

| ID | 要件 |
|---|---|
| B-01 | 章02〜05を CHAPTERS に登録し、§3の t割当表どおりに3D主役・テキスト行を駆動する |
| B-02 | 3D主役は全て初期ロード時に構築し `visible` とt駆動プロパティのみで制御（PER-02） |
| B-03 | テキストは章ごとに `position:sticky` のパネルに行単位で格納し、行の表示は tl 閾値で点灯（class切替。CSSトランジションは0.3s以下） |
| B-04 | 操作語は `IS_TOUCH` で文言差替え（例: 「左クリック」↔「FIREボタン」）。キーボード表記はSPに出さない |
| B-05 | 章04の赤色警告演出は `prefers-reduced-motion` で明滅せず常灯相当にする |
| B-06 | 章03の爆発・章05の同調成立で SoundEngine を薄く発音（AudioContext未解禁時は無音で成立）。**同じtl区間を往復した際の多重発音を防ぐエッジ検出つき**（音のみ例外的に状態を持つ） |
| B-07 | 旧セクション MISSION / TECHNOLOGY の本文はこの4章に吸収し、重複する世界観コピーは削除する |

## 3. 章別詳細設計

> 表記: tl は章ローカル進行度0〜1。3D座標はカメラ相対のステージ位置（カメラ前方 -z）。

### 章02 FLIGHT — 「操縦訓練」 (全体t .18〜.34)

**3D**: `Assets.buildRock()`×12（小型、帯状配置）+ `Assets.buildShip()` 1機
| tl | 3D | テキスト行 |

| 0〜.25 | 岩12個が前方から左右へ視差流過（位置=f(tl)） | 章タイトル `BRIEFING 02 — FLIGHT` グリッチイン |
| .25〜.55 | 機体がフェードイン、カーソル/指位置に±1.2内で追従（ライブデモ・唯一の入力連動） | 行1 |

**コピー**:

### 章03 WEAPONS — 「武装と資源」 (t .34〜.50)

**3D**: `Assets.buildCore()` 1基 + トレーサー円柱2本 + 爆発一式（リング/閃光/破片6個 — ページ専用に構築）+ 標的岩1個
