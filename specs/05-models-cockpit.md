# SPEC 05 — 機体・コア・コックピット造形（Phase 3）

状態: **Verified (2026-07-20 テスト14/14・draw calls +21で予算内・CHASE/COCKPIT目視確認。翼端灯はPointLightでなくスプライト発光で実装=ライト数抑制。コックピット視点では機体を非表示化)** ／ 外部素材: **不要**（プリミティブ合成＋Canvas。任意で宇宙HDRI反射 — SPEC-00 §4）

## 1. 目的

CHASE視点の機体・回収対象のコア・COCKPITの細部を作り込み、
「乗っている/追っている」実在感を高める。ゲームプレイへの影響はゼロに保つ(CON-03)。

## 2. 要件

### 機体（CHASE視点用）
| ID | 要件 |
|---|---|
| SHP-01 | 機体に主翼・垂直尾翼・エンジンノズル×2を追加し、シルエットを戦闘機らしくする |
| SHP-02 | エンジン排気グロー(加算スプライト+短いトレイル)を追加し、長さ/輝度が速度に連動する |
| SHP-03 | 旋回時に翼端灯(左右: 赤/緑)が視認できる。被弾無敵中の点滅挙動は現状維持 |
| SHP-04 | 追加造形は既存のバンク/ピッチ表現・当たり判定に影響しない |

### エネルギーコア
| ID | 要件 |
|---|---|
| COR-01 | コアを「内核(発光八面体)+外周2重リング(異速回転)」構成にする |

| COR-03 | 回収時はコア本体を即消灯し、SPEC-04のスパーク+SPEC-01のcoreGet音で回収感を出す |
| COR-04 | 当たり判定半径・出現数・スコアは不変 |

### コックピット
| ID | 要件 |
|---|---|
| CPT-01 | コンソールにグリーブル(小型ボックス/円筒/配線パイプ 計12〜20個)を追加し情報密度を上げる |

| CPT-03 | 常時明滅する小型インジケータランプ列(シアン/白、3〜5個、非同期周期)を配置する |

## 3. 技術設計

### 3.1 機体（`build()` の ship 構築部を拡張）

- 主翼: 押し出し風に `Shape`+`ExtrudeGeometry`(depth 0.12) または平板Box斜め配置(後退角30°)
- ノズル: `CylinderGeometry(0.5,0.7,1.2,8)` ×2 + 内側に発光ディスク
- 排気: `PlaneGeometry` 加算スプライト2枚重ね。`scale.y = 1 + speed/SPD_MAX*2.2`、
  `opacity = .5 + speed/SPD_MAX*.5`。loop()内で速度連動更新(SHP-02)
- 翼端灯: 小球 MeshBasic 赤(左)/緑(右) + 対応色 PointLight(intensity 2, distance 4)
- 全パーツは `ship` グループ子付け → 既存の傾き表現に自動追従(SHP-04)

### 3.2 コア（`build()` のコア生成部を置換）

```js
core = Group[
  inner: Octahedron(0.9) cyan Additive (既存流用),
  ring1: TorusGeometry(1.5, .06, 6, 32) cyan Additive, rotation.x=.5,
  ring2: TorusGeometry(1.9, .04, 6, 32) white Additive, rotation.z=.9
]
// loop(): ring1.rotation.y += 1.8*dt; ring2.rotation.x -= 1.2*dt(COR-01)
```
- 46個中コアは既存個数のみ。Groupごとプール、userData.r 既存値維持(COR-04)

### 3.3 コックピット（`buildCockpit()` 拡張）

- グリーブル: 共有ジオメトリ3種(Box/Cyl/半円管)×共有マテリアル2種(dark/metal既存流用)を
  乱数配置ではなく**手置き座標表**(コード内配列)で12〜20個配置 — 視界を遮らない下部/側部限定
- ベゼル: 各計器パネルの外周に `BoxGeometry` 額縁4辺(metal) + 内側に1px相当の発光トリム
- インジケータ: 小Plane(0.05)×5 MeshBasic。`loop()` でなくコックピット照明用の
  低頻度タイマー(200ms)で opacity を階段変化(CPT-06のため毎フレーム更新を避ける)
- 非常灯(CPT-04): `redAlert = PointLight(0xff2040, 0, 3)` + ランプ球。
  `hud()` 側で shields<=0 検知時に有効化、`loop()` で `intensity = (sin(t*5.2)+1)*2`。
  reduced-motion時は `intensity=2` 固定(CPT-05)
