# SPEC 08-D — デザイン検討書（連続アクションシークエンス）

状態: **Draft** ／ 親: [SPEC-08](08-action-sequence.md) ／ 実装詳細: 08a/08b/08c

## 1. 演出の基本文法「5拍構成」

全ての説明章は同じリズムで構成する。文法を統一することで、閲覧者は2章目以降
「止まったら読む番だ」と学習し、説明が能動的な行為になる。

```
①アクション(飛行/事件の発生) → ②フリーズ(決定的瞬間で静止+減彩)
→ ③説明(凍結中に行が順に点灯) → ④解除(スクロール続行で時間再開)
→ ⑤結末(爆発完了/シールド砕散などが一気に再生され、飛行へ復帰)
```

- フリーズ区間は章の約40%を占め、**読み終わるまで景色が動かない**ことを保証する
- 解除は閲覧者自身のスクロールで起きる — 「結末を再生するのは自分」という手触りを残す
- 逆スクロールでは ⑤→①へ完全に逆再生される（時間操作もこの世界の物理、として肯定する）

## 2. ビート絵コンテ（章別）

> ta=warp後のアクション進行度 / FZ=フリーズ区間(tl基準) / カメラは機体追従からの相対

## 3. フリーズの視覚言語（ACT-04）

| 要素 | 凍結中の状態 | 実装 |
|---|---|---|
| 色 | 彩度-50%・コントラスト+8%（世界が「止まった」合図） | `#space`へのCSS filter、fz深度でイーズ |
| 動き | 岩・機体・排気・塵がすべて静止（ta停止で自動） | warp写像 |
| HUD | `⏸ TIME HOLD` バッジがhud-status隣に点灯 | fz>0で表示 |
| 音 | 進入時に低いホールド音1拍、解除時に解放音 | `ui('hold')`新設 / `ui('sync')`転用 |
| テキスト | 凍結中のみ行が順点灯（読む時間=凍結時間） | data-on閾値をFZ区間内に割当 |

- 減彩はDOMフィルタで行い3D負荷ゼロ。ブルームは維持（凍結中も発光体は光る=SF的静止）
