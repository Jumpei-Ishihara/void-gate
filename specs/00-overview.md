# SPEC 00 — 全体設計・SDD運用ガイド

素材クオリティ向上（[QUALITY_PLAN.md](../QUALITY_PLAN.md)）をSDD（仕様駆動開発）で進めるための
共通仕様と運用ルール。個別機能の詳細設計は SPEC 01〜06 を参照。

## 0. SDD運用ルール

1. **仕様が先、実装は後**。各SPECは `Draft → Approved → Implemented → Verified` の状態を持つ
2. 実装は **Approved になったSPECのみ** 着手する。仕様変更はSPEC改訂→再承認から
3. 各要件は一意ID（例: `AUD-01`）を持ち、受け入れ基準（AC）と1:1以上で対応する
4. Verified の条件: 受け入れ基準をブラウザ実測（デスクトップ+SP相当ビューポート）で全通過
   ＋コンソールエラー0＋性能予算内
5. 1SPEC=1コミット系列とし、コミットメッセージにSPEC番号を含める（例: `[SPEC-01] 機関砲SFX実装`）

| SPEC | 対象 | Phase | 状態 |
|---|---|---|---|
| 01 | サウンドエンジン/SFX | 1 | Verified |
| 02 | ブルーム(発光)ポストプロセス | 1 | Verified |
| 03 | 小惑星の質感・個体差 | 2 | Verified |
| 04 | 破壊・戦闘エフェクト | 2 | Verified |
| 05 | 機体・コア・コックピット造形 | 3 | Verified |
| 06 | 背景・SIGNAL演出・性能自動調整 | 4 | Verified |

| 07b | └ 訓練章 02〜05（コピー全文含む） | 5-B | Implemented |
| 07c | └ 導入/出口章・HUD/コピー統一 | 5-C | Implemented |

## 1. 共通制約（全SPECに適用）

| ID | 制約 |
|---|---|
| CON-01 | 完全オフライン動作を維持する。外部CDN・外部URLへの実行時アクセス禁止 |
| CON-02 | 追加アセットは原則プロシージャル生成（Canvas/Web Audio/頂点加工）。ファイル同梱する場合もリポジトリ内 `libs/` `assets/` に置きオフラインを維持 |
| CON-03 | 既存のゲームルール・操作・スコア計算を変更しない（本件は「素材の質」のみが対象） |
| CON-04 | 既存のSP対応（横画面・FIREボタン・コンパクトHUD・回転ガイド・フルスクリーン選択）を退行させない |
| CON-05 | ポインター（レティクル）の色・透明度は変更しない（過去のユーザー指示） |
| CON-06 | `prefers-reduced-motion` 時は新規の激しい明滅演出を抑制する |

## 2. 性能予算（全SPECの上限）

| 項目 | デスクトップ | SP(ミドル) |
|---|---|---|
| 目標フレームレート | 60fps | 40fps以上 |
| 1フレームのJS処理 | < 8ms | < 16ms |
| draw call 増分(ゲーム中) | +30以下 | +20以下 |
| 追加メモリ(テクスチャ計) | < 32MB | < 16MB |
| 配布物サイズ増分 | 合計 +300KB以内（外部素材を採用しない場合はほぼ0） |

- 計測方法: `performance.now()` によるフレーム時間ログ＋Chrome DevTools。
  SPはビューポート812×375＋CPU 4x throttleで代替検証し、最終は実機確認

## 4. 外部素材の要否（サマリ）

**結論: 全SPECとも外部素材なしで実装可能**（プロシージャル生成が既定）。
ただし、さらに質を上げたい場合の任意アップグレード素材を以下に明記する。
採用する場合はダウンロードして同梱（CON-01/02）し、ライセンス表記をREADMEに追加する。

| 用途 | 既定(生成) | 任意アップグレード素材 | 入手先例 | ライセンス | 容量目安 |
|---|---|---|---|---|---|
| 小惑星テクスチャ | Canvasノイズ生成 | 岩石PBR(Color/Normal 512px) | ambientCG (Rock系) | CC0 | 300KB〜1MB |
| 爆発/射撃SE | Web Audio合成 | 実録SEサンプル(ogg) | freesound.org (CC0検索) | CC0のみ採用 | 50〜200KB |
| 環境反射(機体) | なし(emissiveで代替) | 宇宙HDRI(1K) | Poly Haven | CC0 | 500KB〜1.5MB |
| BGM | 合成ドローン(SPEC-01) | ループBGM(ogg) | 要制作 or CC0曲 | 要確認 | 1〜3MB |

- **CC0以外（CC-BY等）は帰属表記が必要になるため原則不採用**。採用時はREADMEにクレジット必須
- BGMは容量インパクトが大きいため既定では合成ドローンのみ。実曲が欲しい場合は別途相談

## 5. 検証環境

- デスクトップ: Chrome最新（プレビューブラウザ）
- SP相当: ビューポート812×375（横）/ 375×812（縦・回転ガイド確認）
- 実機: ユーザー所有のスマートフォン（GitHub Pages経由）
- リグレッション確認: 既存チェックリスト＝ゲーム起動/終了、視点切替、ロックオン、
  被弾4回で大破、シールドCRITICAL表示、FIREボタン、フルスクリーン選択、SIGNAL 5ch同調
